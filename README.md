# delaySystem
Demo System for response time delays.

- v1 = No delay.
- v2 = 3 second delay.
- v3 = 1 second delay.

See branch `v1`, `v2` & `v3` for code.
`index.html` is just visual. Delay is coded into `express.js`

## Build & Push Example
```
docker build -t adamgardnerdt/delaydemosystem:v1
docker push
```
